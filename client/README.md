# imbox-client

## Requirement

**Building**

  * JDK 1.6+ (Java 8 recommended)
  * [Maven](https://maven.apache.org/) 

**Executing**

  * JRE 1.6+ (Java 8 recommended)

## Usage

**Building**

Make sure you are in the *imbox/client/* .

    > mvn clean compile assembly:single

**Executing**

	> java -jar target/Imbox-xxx.jar
